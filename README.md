# Task-Management-Tracker(Excel)

A simple and effective Task Management Tracker built using Microsoft Excel to help individuals and teams plan, track, and monitor tasks efficiently. This tracker enables users to manage tasks with clear visibility on status, priority, deadlines, and ownershipâ€”without the need for complex tools.

**Key Features:**
- Task creation and categorization
- Priority and status tracking (Not started, In Progress, Completed)
- Due date monitoring
- Task Owner/assignee tracking
- Progress tracking using formulas and conditional formatting
- Easy-to-use, lightweight, and customizable Excel template

**Use Cases:**
- Project coordination
- Personal task tracking
- Small team task management
- PMO and QA task tracking

**Tools Used:**
- Microsoft Excel (Formulas, Conditional Formatting, Data Validation)

This project demonstrates practical skills in task organization, process tracking, and reporting using Excel.
